## Operating Rules

1. Capture into [[📥 Inbox/Quick Capture]].
2. Convert actionable work into a task with a due date and project tag.
3. Convert durable insight into an atomic knowledge note.
4. Link research and experiments back to the project they serve.
5. Review weekly; prune monthly; archive completed work.
