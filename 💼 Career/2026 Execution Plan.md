2. **Turn AOI into a Flagship Project**: Establish it as a defensible portfolio piece showcasing observability.

## Month-by-Month Plan

### July 2026
* **Primary Outcome**: Complete thesis report and submit all graduation deliverables.
* **Secondary Outcomes**:
  1. Finalize the backend API for the `AOI` project (FastAPI, structured JSON logs).
  2. Set up local Loki/Promtail/Grafana integration for `AOI` to verify latency monitoring.
  3. Ensure the `HASC` platform has basic Docker configurations working locally.
* **Evidence of Completion**: Submitted thesis draft, local `AOI` repository showing Grafana dashboard screenshots.
* **Do NOT Start**: Cloud deployments, AWS architectures, or studying for Kubernetes certifications.

### August 2026
* **Primary Outcome**: Defend thesis successfully and secure graduation approval.
* **Secondary Outcomes**:
  1. Gather all official academic transcripts, letters of recommendation placeholders, and degrees.
  2. Optimize the `SilentVoix` sensor glove firmware to output stable real-time IMU and flex sensor data.
  3. Write a polished resume and portfolio landing page showcasing your four core projects.
* **Evidence of Completion**: Signed thesis defense evaluation sheet, clean resume PDF, live portfolio landing page on GitHub Pages.
* **Do NOT Start**: AWS exam prep or Terraform coding.

### September 2026
* **Primary Outcome**: Apply and prepare for VinAI AI Real-World program or local Junior MLE/Backend roles.
* **Secondary Outcomes**:
  1. Practice 1 algorithmic problem per day on LeetCode (focus on Arrays, Trees, HashMaps).

  3. Clean up raw data pipelines for `TempCastML` room temperature forecasting.

### October 2026
