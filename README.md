## Operating Loop

1. Capture into [[📥 Inbox/Quick Capture]].
2. Plan and execute from project workspaces.
3. Convert durable insight into connected knowledge notes.
4. Record experiments as hypotheses, configurations, results, and decisions.
5. Review weekly and monthly; update goals and project state.

## Workflow Plugins

- Dataview for live indexes and dashboards
- Tasks for commitments and deadlines
- Templater for schema-safe note creation
- Kanban for project flow

Markdown remains canonical. Plugins are replaceable views over the same files.
